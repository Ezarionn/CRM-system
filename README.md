# CRM-system

<h3 style="font-size: 24px;">Система управления контактными данными клиентов.</h3>

<h2>Описание</h2>
<h3>Приложение позволяет работать с контактной информацией клиентов и предоставляет следующие возможности:
- Просмотр списка клиентов в виде таблицы;
- Добавление новых клиентов;
- Изменение информации о существующем клиенте;
- Удаление клиентов;
- Сортировка клиентов по полю;
- Поиск клиента
</h3>

<h2>Разработка</h2>
<h3>Для создания сайта использованы HTML5, CSS3 и JavaScript. HTML отвечает за статическую структуру сайта, в то время как динамическая отрисовка таблицы с данными осуществляется с помощью JavaScript. Также были использованы сторонние библиотеки, такие как Choices.js и Tippy.js, для решения специфических задач и улучшения пользовательского интерфейса.</h3>

<h2>Запуск</h2>
<h3>Frontend: перейдите в папку с фронтендом и откройте файл `index.html` с помощью Live Server.</h3>
<h3>Backend: перейдите в папку с бэкендом и введите команду 'node index'.
После запуска сервера API будет доступен по пути `http://localhost:3000`.</h3>
